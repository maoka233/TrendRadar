┌─────────────────────────────────────────────────────────┐
│ 四、报告内容说明                                         │
└─────────────────────────────────────────────────────────┘

每个热点话题显示：

  [微博] 🆕 OpenAI发布新模型 [1-3] - [10时30分~15时30分] (5次)
  ↓        ↓                  ↓        ↓                    ↓
  平台   新增标记           排名历史  持续时间            出现次数

【图标说明】
  🆕 = 新增话题（首次出现）
  🔥 = 高热度话题（≥10条）
  📈 = 中热度话题（5-9条）
  📌 = 普通热度（<5条）

【排名说明】
  [1-3] = 排名在1到3之间变化（红色高亮）
  [7]   = 固定在第7名（普通显示）

【时间说明】
  10时30分 = 仅出现一次
  [10时30分~15时30分] = 持续出现（首次到最后）

┌─────────────────────────────────────────────────────────┐
│ 六、常见问题                                             │
└─────────────────────────────────────────────────────────┘

Q1: 为什么没有评论内容？
A1: API不提供评论，点击话题链接可跳转查看原文评论

Q2: 如何恢复到原来的11个平台？
A2: 编辑 config/config.yaml，添加其他平台ID

Q3: 如何修改关键词？
A3: 编辑 config/frequency_words.txt，用空行分隔不同主题

Q4: 热点太多/太少怎么办？
A4: 调整关键词的精确度，使用必须词(+)或过滤词(!)

Q5: 如何查看更多平台？
A5: 访问 https://newsnow.busiyi.world/ 查看支持的平台
